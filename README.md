# Social-Media-Addiction-Self-Regulated-Learning-Academic-Procrastination-A-SEM-PLS-Replication-Study
This repository contains the data, instrument, and findings from a quantitative research project replicating Nurmaidah et al. (2025), investigating the relationship between Social Media Addiction (X), Self-Regulated Learning (Z), and Academic Procrastination (Y).
Overview

Design: Cross-sectional survey
Sample: 160 respondents (school, college, and university students)
Instrument: 17-item, 5-point Likert scale questionnaire (5 items per construct + demographics)
Analysis tool: SmartPLS 4.1 (SEM-PLS)

Variables
VariableRoleDescriptionSocial Media AddictionIndependentExcessive/compulsive social media use disrupting daily lifeSelf-Regulated LearningMediatorPlanning, monitoring, and controlling one's own learning processAcademic ProcrastinationDependentVoluntary delay of academic tasks despite anticipated negative consequences
Key Findings

✅ H1 Supported — Social Media Addiction → Academic Procrastination (β = 0.292, T = 3.066, p = 0.002)
❌ H2 Not Supported — Social Media Addiction → Self-Regulated Learning (β = 0.045, p = 0.621)
❌ H3 Not Supported — Self-Regulated Learning → Academic Procrastination (β = -0.025, p = 0.833)
❌ H4 Not Supported — Mediation via Self-Regulated Learning (indirect β = -0.001, p = 0.927)

Only the direct path between social media addiction and procrastination was statistically significant; the proposed mediating role of self-regulated learning was not confirmed in this more heterogeneous sample.
Tech/Methods
SEM-PLS · Bootstrapping (5,000 subsamples) · Construct reliability (Cronbach's Alpha, Composite Reliability) · Convergent validity (AVE) · Discriminant validity (HTMT) · Collinearity (VIF)
